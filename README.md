## Démarrage

Ce projet utilise Vite + React + TypeScript. Les instructions ci-dessous expliquent comment initialiser et lancer l'application en environnement de développement, ainsi que les commandes de build et de preview.

- **Prérequis**: Node.js (version 16+ recommandée) et un gestionnaire de paquets `npm`.
- **Port par défaut**: Vite démarre généralement sur le port `5173`.

### 1) Cloner le dépôt

```bash
git clone https://github.com/thomasduppi/react-movie-finder.git
cd react-movie-finder
```

### 2) Installer les dépendances

Avec npm:

```bash
npm install
```

### 3) Lancer en développement

```bash
npm run dev
```

Cette commande démarre le serveur de développement. Ouvrez `http://localhost:5173` (ou l'URL affichée dans la console).
